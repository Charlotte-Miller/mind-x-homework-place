1. How to check a variable's value?

To check a variable's value, we could use typeof operator.
For examples:
console.log(typeof 22);
//Expect output: 'number'.

2. In what cases, you will get SyntaxError from the compiler telling you that some of your variables have invalid names ? Can you give 3 different examples of invalid names?

There are some rules when constructing name for variable:

  • Names can contain letters, digits, underscores, and dollar signs.
  • Names must begin with a letter, $, or _.
  • Names are case sensitive (y and Y are different variables)
  • Reserved words (like JavaScript keywords) cannot be used as names
Any cases constructing variable name breaking these rules, the compiler will give you a SyntaxError of invalid name.

For example:
  • let name*;
  • const let = 6;
  • const 3pigs = 3;
