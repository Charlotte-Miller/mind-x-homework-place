1. What is Boolean? Write down 3 different expression that results a Boolean type.

Boolean is a logical data type that can have only the values true or false.

    For example:
  • console.log(5 === 5); //Print: True;
  • console.log(5 >= 6); //Print: False;
  • console.log('a' === 'b'); //Print: False;
